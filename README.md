# Muhammad Zia ul Hassan — Portfolio Website

A personal portfolio website for an aspiring Machine Learning Engineer, built with pure HTML, CSS, and vanilla JavaScript. No frameworks, no dependencies — just clean, fast, and deployable anywhere.

## 🌐 Live Demo

[https://ziaulhassan87.github.io/zia](https://ziaulhassan87.github.io/zia)

---

## 📋 Features

- Fully responsive design (mobile, tablet, desktop)
- Mobile hamburger navigation with smooth slide-down drawer
- Animated hero section with rotating photo frame
- Fade-up scroll animations using Intersection Observer
- Projects, Skills, Certifications, and Contact sections
- Fixed glassmorphism navbar with backdrop blur
- Dark navy theme with blue accent colors
- Zero dependencies — no npm, no build step

## 🚀 Deployment (GitHub Pages)

1. Create a GitHub repository named `zia`
2. Upload `index.html` and `preview.jpg`
3. Go to **Settings → Pages**
4. Set source to `main` branch, root folder
5. Site will be live at `https://yourusername.github.io/zia`
